Module 2 workshop pack — drop files here
=========================================

This folder is where the downloadable workshop pack for Module 2 lives.

Expected files (referenced in src/content/modules/02-public-repos.md):

  FAIR_RNAseq_Facilitator_Guide.pdf
  FAIR_RNAseq_Workshop_Slides.pdf

When real files are added with those names, the "Workshop pack" section
on the Module 2 page lights up with working download links.

If a file is renamed, update the `file:` paths in the module's
`workshopAssets:` frontmatter to match.

This README file does not appear anywhere on the site — it's just here
to make the folder's purpose obvious.

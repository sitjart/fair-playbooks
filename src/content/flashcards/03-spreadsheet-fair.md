---
question: "A PI hands you a spreadsheet of bespoke experimental data with no clear repository home. Name three things you'd ask for before depositing it on Zenodo."
category: reusable
difficulty: 2
relatedModule: 03-no-standards
---

1. **A README** in plain language describing what each file is.
2. **A data dictionary** explaining what each column means, units, and allowed values.
3. **An explicit license** (CC-BY 4.0 is a safe default for data).

Bonus: convert XLSX to CSV per sheet for long-term openness.

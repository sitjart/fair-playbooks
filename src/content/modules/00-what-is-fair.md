## The four principles

- **Findable** — data and metadata are described in a way that humans and machines can locate them
- **Accessible** — once found, data can be retrieved using clear protocols (this does not mean "open")
- **Interoperable** — data uses shared vocabularies and formats so it can be combined with other data
- **Reusable** — data is well-described, licensed clearly, and meets community standards

## Common misconceptions to surface

- "FAIR means open" — no, it means *clear about how to access*
- "FAIR is just for big repositories" — applies to your spreadsheet too
- "FAIR is a checklist" — it's a set of principles, implemented through judgment and tools
